Introduction to Candy

Published on June 1, 2025

Candy is a sweet and simple info gathering tool designed to collect Telegram sessions, Discord data, screenshots, and system info efficiently.

Features

Telegram Session
Discord Account (Token, E-mail, Phone)
Screenshot + System Info

Installation

System Requirements

  • Python 3.12.x
  • Git
  • Windows 10-11
  • RAM 8+
